產生 ERROR Template render error 原因

當執行hexo g後出現 ERROR Template render error，大致上可能是這三個原因，都會讓Hexo在編譯時候就會爛掉

  1. 在.md 或 swig模板中誤用{ { 及 } }這符號

  2. 在swig模板中有include某個不存在的swig

    {% if theme.recommended_posts.enabled and not is_index %}
  
    
	{% include 'recommended_posts.swig' with {post: post, site: site} %}
  
    
{% endif %}
{%- if theme.related_posts.enable and (theme.related_posts.display_in_home or not is_index) %}
 {{ partial('_partials/post/post-related.swig') }}
{%- endif %}
  1. 在.config.yml中配置某個設定
